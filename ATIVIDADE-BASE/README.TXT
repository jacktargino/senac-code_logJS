📚 Sumário dos Assuntos – Atividade JavaScript

✅ Declaração de Variáveis
const – Constantes e imutabilidade

let – Variáveis com escopo de bloco

var – Variáveis com escopo de função


🔎 Manipulação de Elementos DOM HTML
getElementById – Captura de elementos da página

innerText – Exibição de conteúdo na interface


🧠 Condicionais
if e else – Decisão binária (aprovado/reprovado)

if, else if e else – Múltiplos caminhos de decisão


🔁 Laços de Repetição
do while – Execução garantida pelo menos uma vez

while – Repetição baseada em condição

for – Contador com controle total do índice


🧭 Controle de Fluxo
switch case – Avaliação de múltiplos casos com break


🧮 Funções
arrow function – Funções modernas e compactas

array com split() e for – Manipulação de listas de dados


💬 Exibição e Depuração
alert() – Exibir mensagens simples em pop-up

console.log() – Registro de mensagens no console do navegador
